RseQC
The software includes infer_experiment.py ==> can infer strandness from RNA-seq reads
Mapping your RNA-seq reads to the selected reference genome as if they were non-strand specific using Bowtie2
The alignment is then compare to reference annotation to infer the strandness of reads
1.For non-strand specific experiment the ratio of explanations should be close to 50:50.
2.For strand specific experiments there are two scenarios:
(1)Reads in file 1 are always on the same strand as the gene;
(2)Reads in file 2 are always on the same strand as the gene.
(RSeQC: An RNA-seq Quality Control Package)
