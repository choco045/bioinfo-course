2. 统计IV号染色体上各类feature（第3列）的数目，并按升序排列

- 命令：
```bash
awk -F'\t' '$1=="IV" {print $3}' 1.gtf | sort | uniq -c | sort -n
```

- 输出：
```
    853 start_codon
    853 stop_codon
    886 gene
    886 transcript
    895 CDS
    933 exon
```

3. 寻找不在IV号染色体上的所有负链上的基因中最长的2条CDS序列，输出它们的长度

- 命令：
```bash
awk -F'\t' '$1!="IV" && $7=="-" && $3=="CDS" {len=$5-$4+1; print len}' 1.gtf | sort -rn | head -2
```

- 输出：
```
14730
12276
```

4. 寻找XV号染色体上长度最长的5条基因，输出基因id及对应的长度

- 命令：
```bash
awk -F'\t' '$1=="XV" && $3=="gene"{len=$5-$4+1; split($9,a,";"); name = a[1]; gsub(/gene_id |"/, "", name); print name"\t"len}' 1.gtf | sort -k 2 -nr | head -5
```

- 输出：
```
YOL081W 9240
YOR396W 5391
YOR343W-B       5314
YOR192C-B       5314
YOR142W-B       5269
```

5. 统计 `1.gtf` 的列数

- 命令：
```bash
grep -v '^#' 1.gtf | awk -F'\t' '{print NF}' | uniq
```

- 输出：
```
9
```
