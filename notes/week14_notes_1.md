# Machine Learning Basics

## Brief Introduction

根据预测变量是否已知，机器学习问题通常可以分为两类：

* **Supervised Learning** ( 监督学习): 模型有明确的输入（自变量/特征）和输出（因变量/响应变量）。如果目标变量（要预测的变量）是类别信息（例如正/负），该问题称为分类问题。如果目标变量是连续的（例如身高）则为回归问题。
* **Unsupervised Learning** (无监督学习): 目标变量是未指定的。模型的目的是挖掘样本点之间的关系。常见的无监督学习任务包括降维和聚类。

我们这里主要聚焦于有监督学习，分5个小节介绍在实践中如何解决一个分类问题：

### 1.1 数据预处理 (Data Pre-processing)
数据预处理的目的是消除原始数据的“噪音”和量纲差异，为模型训练打好基础。

* 缺失值处理：你可以选择剔除样本/特征，直接使用缺失值（适合树模型），或者利用插补法（如均值、K最近邻、回归插补等）填补空缺。

* 数据 Scaling (归一化)：这是为了防止数值大的特征在计算距离（如逻辑回归、SVM）时占据主导地位，或防止模型训练不收敛。常用的方法包括 Z-score（适用于正态分布）和 Robust scaling（适用于有离群值的数据）。

### 1.2 数据可视化与降维 (Visualization & Dimension Reduction)
由于实际数据维度很高，我们需要通过降维将数据投影到二维平面，以直观查看样本间的相似度。

* PCA：线性降维方法，通过寻找最大化方差的线性组合来可视化数据，常用于数据初步分析。

* t-SNE：擅长保持样本的“非线性局部结构”，非常适合处理单细胞转录组等样本间关系复杂的数据。

### 1.3 特征工程 (Feature Extraction and Selection)
常言道“数据和特征决定了机器学习的上限”，特征工程是提升模型效果的关键。

* 特征提取 (Extraction)：通过领域知识将原始数据转化为更有意义的新特征（例如用身高和体重计算 BMI 指数），从而优化模型输入。

* 特征选择 (Selection)：旨在剔除无关特征，降低过拟合风险并减少计算量。

  * Filter (过滤法)：基于统计指标（如 p 值、互信息）直接评分筛选，不依赖模型训练。

  * Embedded (嵌入法)：在模型训练过程中自动评估特征贡献（如随机森林的特征重要性）。

  * Wrapper (包装法)：利用机器学习模型进行交叉验证，通过递归等启发式搜索寻找最优特征组合。
