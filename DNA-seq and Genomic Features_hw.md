## 1. 人类基因组的大小以及基本组成是哪些？

**【解答】**

**人类基因组的大小：**
人类基因组的大小约为 **3.2 Gbp** (Billion base pairs，十亿碱基对)。根据最新的人类基因组参考序列（如 GRCh38.p14），大小约为 3.2 Gbp，而最新的端粒到端粒（T2T-CHM13）无间隙人类基因组组装大小约为 **3.055 Gbp**。

**基因组的基本组成：**
根据讲义及相关权威文献（如 *Nature Reviews*, Alexander et al., 2010），虽然人类基因组极其庞大，但负责编码蛋白质的区域只占很小一部分，其主要组成比例如下（数据由于检测手段进步会有略微浮动，但大致分布类似）：

| 基因组组成成分 | 占比估值 | 说明 |
| :--- | :---: | :--- |
| **Coding regions (蛋白质编码区)** | **~3%** | 真正转录并翻译成蛋白质的序列（外显子为主），约包含两万多个蛋白质编码基因。 |
| **Introns (内含子)** | **~42%** | 存在于基因内部，转录后在RNA剪接过程中被去除的非编码区域。 |
| **Repeats and Transposons (重复序列与转座子)** | **~34%** | 包含各种串联重复序列、散在重复序列（如Alu元件、LINEs、SINEs等），被认为是基因组进化中的“跳跃基因”。 |
| **Regulatory Elements (调控元件)** | **~10%** | 包括启动子、增强子、绝缘子等，负责调控基因在何时、何地以及以何种水平表达。 |
| **Others (其他区域)** | **~11%** | 其他非编码区域及未明确功能的结构序列（曾被称为“垃圾DNA”，但ENCODE计划揭示其中广泛存在非编码RNA的转录活动）。 |

*数据来源参考：Alexander et al., Nature Reviews, 2010 及课程讲义。*

## 2. 基因中的非编码 RNA的最新注释是多少个了？请详细列一下其中的非编码 RNA 的细分类型的数目，并对主要的非编码 RNA 是做什么的用1-2句解释一下。

**【解答】**

**最新非编码RNA注释数目：**
根据最新的人类基因组注释计划（以 **GENCODE Release 47** 为例，基于 GRCh38.p14，更新于 2024 年末左右），注释的非编码 RNA 基因总数已大幅增加，其中长链非编码RNA和各类小分子非编码RNA总计超过 **27,000** 个。

**细分类型数目（GENCODE Release 47 大致数据）：**
* **Long non-coding RNA (lncRNA) genes:** 约 **19,956** 个
* **Small non-coding RNA genes:** 约 **7,489** 个 (其中包含了 miRNA, snRNA, snoRNA, rRNA 等各种小分子的注释)

**主要非编码RNA类型及功能简介：**

1. **rRNA (核糖体 RNA, Ribosomal RNA)**
   * **功能**：与核糖体蛋白组装成核糖体，是细胞内蛋白质合成（mRNA翻译）的机器与核心场所。
2. **tRNA (转运 RNA, Transfer RNA)**
   * **功能**：在蛋白质合成过程中，负责识别 mRNA 上的密码子，并将相对应的氨基酸搬运到核糖体上。
3. **snRNA (核内小分子 RNA, Small nuclear RNA)**
   * **功能**：主要存在于细胞核中，与特定蛋白质结合形成剪接体（Spliceosome），参与 pre-mRNA 中内含子的剪接去除。
4. **snoRNA (核仁小分子 RNA, Small nucleolar RNA)**
   * **功能**：主要定位于核仁，指导 rRNA 和其他小 RNA 靶标的化学修饰（如 2'-O-甲基化和假尿嘧啶化）以及前体 rRNA 的切割加工。
5. **miRNA (微小 RNA, MicroRNA)**
   * **功能**：长约 21-23 nt，通过与靶标 mRNA 的 3' UTR 等区域碱基互补配对，引导 mRNA 降解或抑制其翻译，从而在转录后水平负调控基因表达。
6. **piRNA (Piwi 相互作用 RNA, Piwi-interacting RNA)**
   * **功能**：主要在生殖细胞中高表达，通过与 Piwi 蛋白家族结合形成复合物，沉默转座子的活性，以保护生殖细胞基因组的完整性与稳定性。
7. **lncRNA (长链非编码 RNA, Long non-coding RNA)**
   * **功能**：长度大于 200 nt 的一类异质性 RNA 分子，它们不编码蛋白，但在表观遗传、转录及转录后等各个调控层面上广泛参与基因表达的精细调控（如著名的 *XIST*, *HOTAIR* 等）。
